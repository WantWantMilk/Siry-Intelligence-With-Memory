# Siry - AI Chatroom

A sleek, modern, and privacy-aware chat interface for the AI assistant "Siry" (Silly Harmony), deployed directly on GitHub Pages. It features a glass-morphism design, an access gate, and cross-browser compatibility.

## âœ¨ Features

*   **Modern Glass-Morphism UI:** Features a translucent, blur-background design with smooth animations.
*   **Access Control:** Implements a two-step PIN verification system (short answer + long PIN) with browser fingerprinting for returning users.
*   **Multi-Browser Support:** Automatically detects legacy browsers (like IE) and redirects to a simplified compatibility page.
*   **Mobile-First & Responsive:** Fully responsive design that works flawlessly on both mobile devices and desktops.
*   **Legal Compliance:** Includes a mandatory disclaimer modal regarding AI-generated content and AGPL-3.0 licensing.
*   **Pure Frontend:** Runs entirely on the client-side using static HTML, CSS, and JavaScript.

## ğŸ› ï¸ Tech Stack

*   **Frontend:** HTML5, CSS3 (with Flexbox/Grid), Vanilla JavaScript (ES6+)
*   **API Integration:** SiliconFlow Chat Completions API
*   **Deployment:** GitHub Pages
*   **Styling:** Custom CSS with CSS Variables, `backdrop-filter` for glass effect

## ğŸš€ Quick Start

1.  **Clone or Download:** Clone this repository or download the `index.html` and `ie.html` files.
2.  **Configure API:**
    *   Open `index.html` and `ie.html`.
    *   Locate the `CONFIG` object (in `index.html`) or `API_KEY` & `MODEL` variables (in `ie.html`).
    *   **Crucial:** Replace the placeholder values with your valid **SiliconFlow API Key** and desired **Model Name**.
    ```javascript
    // In index.html
    const CONFIG = {
        API_KEY: 'sk-your-actual-siliconflow-api-key-here', // REPLACE THIS
        MODEL: 'THUDM/GLM-4-9B-0414', // REPLACE IF NEEDED
        // ...
    };
    ```
3.  **Deploy to GitHub Pages:**
    *   Create a new public repository (e.g., `siry-chat`).
    *   Upload both `index.html` and `ie.html` to the root of the repository.
    *   Go to **Settings > Pages**, set the source branch to `main` (or `master`), and save. Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## âš™ï¸ Configuration & Customization

*   **Personality Prompt:** Modify the `SYSTEM_PROMPT` inside the `CONFIG` object in `index.html` to change Siry's character and conversation style.
*   **Access PIN:** The short answer is case-insensitive `"harmory"`. The long PIN is defined in the `LONG_PIN` variable.
*   **Styling:** Tweak colors, blur intensity, and roundness by modifying the CSS custom properties (`:root` variables) in the `<style>` section of `index.html`.

## âš ï¸ Important Notes & Disclaimers

*   **API Key Exposure:** This project embeds the API key **directly in the frontend code**. This is **NOT secure for public production use** and is intended only for personal/trusted projects. Anyone with access to the webpage can view and potentially misuse your key.
*   **AI-Generated Content:** All responses are generated by a third-party AI model and may be inaccurate, biased, or inappropriate. Use at your own discretion.
*   **Privacy:** Conversations are processed through the SiliconFlow API. No chat history is stored on our server, but it may be subject to the API provider's logging policies.
*   **Browser Fingerprinting:** The "returning user" feature uses a simple fingerprint based on browser properties, not a true IP address. It is not a robust security measure.
